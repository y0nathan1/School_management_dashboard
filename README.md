# School Management System (Portfolio Project)

A simple **School Management System** built using **vanilla HTML, CSS, and JavaScript**. It stores students and classes locally and allows you to add/remove records instantly.

---

## ✅ Features

- Add, edit, and remove students (name, grade, class)
- Add, edit, and remove classes (name, teacher)
- Live search + filters
- Dashboard stats (total students / classes)
- Import / export JSON data
- Reset dataset with one click
- Data persists using **localStorage**
- Clean, responsive UI

---

## 🚀 How to Run

### 1) Open in a browser
For best results, serve it from a local server (recommended):

#### Option A (recommended): Use VS Code Live Server
- Install the **Live Server** extension in VS Code.
- Right-click `index.html` and choose **Open with Live Server**.

#### Option B: Use Python (if installed)
```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`

---

## 📝 Notes
- Data is stored locally via your browser’s `localStorage`.
- Clearing browser storage will reset the app.
- Want more features? Consider adding search, sorting, or a backend API.
